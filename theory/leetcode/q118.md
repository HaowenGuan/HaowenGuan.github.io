## Pascal's Triangle
Given numRows, generate the first numRows of Pascal's triangle.


For example, given numRows = 5, return:

```
[
     [1],
    [1,1],
   [1,2,1],
  [1,3,3,1],
 [1,4,6,4,1]
 ]
```
Constraints:

* $1 <= numRows <= 30$

```Python
class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        l = [[1]]
        if numRows == 1:
            return l
        l.append([1,1])
        while numRows > len(l):
            l.append([1])
            for i in range(len(l) - 2):
                l[-1].append(l[-2][i] + l[-2][i + 1])
            l[-1].append(1)
                
        return l
```
