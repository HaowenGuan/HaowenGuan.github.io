## My Answer
```Python
class Solution:
    def reverseBits(self, n: int) -> int:
        result, bit = 0, 0
        while n:
            n, mod = divmod(n, 2)
            if mod:
                result += 2 ** (31 - bit)
            bit += 1
        return result
```

3. Space complexity: O(1)
