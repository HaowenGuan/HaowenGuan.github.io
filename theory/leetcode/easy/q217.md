## My Answer
```Python
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        seen = set()
        for i in nums:
            if i in seen:
                return True
            seen.add(i)
        return False
```

### Interpret
1. Hashset
2. Time complexity: O(n)
3. Space complexity: O(n)
